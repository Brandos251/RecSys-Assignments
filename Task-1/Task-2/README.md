## Цель
Реализовать **кластеризацию пользователей** на основе матрицы косинусных подобий.  
Задача заключается в поэтапном объединении пользователей в кластеры, если их сходство превышает заданный порог **R**.

## Матрица расстояний
Матрица задает косинусное подобие между пользователями:

|    | U1   | U2   | U3   | U4   | U5   |
|----|------|------|------|------|------|
| U1 | 0    | 0.88 | 0.93 | 0.87 | 0.87 |
| U2 | 0    | 0    | 0.88 | 0.85 | 0.86 |
| U3 | 0    | 0    | 0    | 0.88 | 0.86 |
| U4 | 0    | 0    | 0    | 0    | 0.83 |
| U5 | 0    | 0    | 0    | 0    | 0    |

- Столбцы и строки — пользователи (U1–U5)  
- Значения — косинусное подобие между пользователями  
- Порог кластера: **R = 0.85**

## Алгоритм работы
1. На каждом шаге **находим максимальное значение** в матрице.  
2. Если максимальное значение ≥ R, объединяем соответствующих пользователей в кластер.  
3. Пересчитываем матрицу по **методу ближайшего соседа**:
   - Для нового кластера берется максимальное значение косинусного подобия с другими пользователями.  
4. Удаляем объединенного пользователя из матрицы и обновляем словарь пользователей.  
5. Повторяем шаги, пока **нет значений ≥ R**.
