## Цель
Реализовать **подбор подходящих пользователей и предсказание оценки товара** для пользователя на основе матрицы предпочтений, используя косинусное сходство.

Задача фокусируется на:

1. Выборке пользователя «а», для которого требуется предсказать оценку.
2. Исключении пользователей с неполными данными по выбранному продукту.
3. Построении матрицы схожести (косинусного подобия) между оставшимися пользователями.
4. Определении ближайших пользователей по схожести и использовании их оценок для рекомендации.
